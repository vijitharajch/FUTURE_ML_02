# FUTURE_ML_02
# 🎫 AI Support Ticket Classifier

An AI-powered NLP application that automatically classifies customer support tickets and predicts their priority using Machine Learning and Streamlit.

---

# 🚀 Features

- Support Ticket Classification
- Priority Prediction
- NLP Text Preprocessing
- TF-IDF Vectorization
- Machine Learning Models
- Streamlit Web Application
- Real-Time Predictions

---

# 🧠 Machine Learning Models Used

- Logistic Regression
- Naive Bayes
- Support Vector Machine (SVM)
- Random Forest
- XGBoost

---

# 🛠️ Technologies Used

- Python
- Scikit-learn
- NLP
- TF-IDF
- Streamlit
- Pandas
- NumPy

# ⚙️ Installation

Clone the repository:

```bash
git clone https://github.com/yourusername/AI-Support-Ticket-Classifier.git
```

Move into project directory:

```bash
cd AI-Support-Ticket-Classifier
```

Install dependencies:

```bash
pip install -r requirements.txt
```

Run the Streamlit application:

```bash
streamlit run app.py
```
